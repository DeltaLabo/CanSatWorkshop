# PCB General Rules

- Minimum track width: 0.8 mm
- Minimum Via Diameter: 1.2 mm
- Minimum Via Drill: 0.8 mm
- Minimum Pad Diameter: 75 mil
- Minimum Pad Drill Diameter: 45 mil
- Wire-to-Wire Spacing: 0.8 mm

- Place WCS Origin on a corner of the top stock surface
- Use a Bore operation for pin holes of a diameter that is not available in the tool set
- Use a 2D Contour operation for tracks
- Use a Ramp operation for the outer border
- Use a Drill operation for holes that exactly match available tool sizes
