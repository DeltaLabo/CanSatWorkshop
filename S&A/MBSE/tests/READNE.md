# S&A MBSE testing plan

## Scope

This plan is derived from the Capella-style D2 Physical Architecture views in `S&A/MBSE`:

- `v0.1/SAA_v0.1_view1_physical.d2` — structural module stack, backplane volume, parachute/top module, and S&A constraints before PCB mounting.
- `v1.0/SAA_v1.0_view1_physical.d2` — integrated assembly with ADS/AMS/OBCC/PDS PCBs, ESS battery pack, backplane PCB, headers, PDM servo connector, cap, parachute, and servo.

The Capella PA interpretation used here is: `[PC]` nodes are physical components, `[PL]` edges are physical links/fixtures, and `[C]` notes are verification constraints. The diagrams are not edited by this plan.

## Statistical policy

- **Attribute/reliability tests:** for zero-failure demonstration, use the binomial rule `n >= ceil(log(1-C)/log(R))`, where `R` is target reliability and `C` is confidence. Default qualification target: **R90/C90 = 22 independent successful trials with 0 failures**. Higher-criticality target: **R95/C90 = 45 independent successful trials with 0 failures**. If a workshop cannot afford this, a smaller run is a screen only, not a reliability claim.
- **Variable measurements:** use calibrated instruments and guard bands. A measured value passes only if the worst-case value plus/minus expanded uncertainty stays inside the modeled limit.
- **Communication tests:** with zero errors over `N` frames, the one-sided 90% upper bound on per-frame error probability is approximately `2.3/N`.
- **Repeated tests on the same article** show robustness/fatigue of that article, but do not count as independent population samples unless separate representative articles are used.

| SAA-T01 | Integrated CanSat fits the modeled 355 ml can envelope before launch/deployment. | Expected use case | v1.0 CanSat assembly, fully integrated except external antennas if applicable. | Model `envelope_limits`; Crown 12 oz/355 ml can reference; ESA/Andøya CanSat size guidance. | Measure max diameter and total height at multiple clock angles and stack locations using calibrated calipers / height gauge. | Minimum 3 repeated measurements per dimension; use worst value with uncertainty. | Diameter + U95 ≤ 66.0 mm and height + U95 ≤ 122.0 mm per model. | SAA-T02 | Integrated CanSat mass remains below the CanSat limit. | Expected use case | v1.0 CanSat assembly in as-flown configuration. | Model `envelope_limits`; ESA/Andøya 300–350 g CanSat mass guidance. | Weigh fully integrated assembly, including battery, parachute hardware, fasteners, and ballast if used. | Minimum 10 stable scale readings, calibrated scale resolution ≤0.1 g; use mean and U95. | Mean mass + U95 < 350.0 g. | SAA-T03 | Physical architecture matches the MBSE model. | Expected use case | v0.1 module set and v1.0 integrated assembly. | PM&SE IVV rule: inspect every modeled physical component and physical link. | Presence/absence inspection of every `[PC]` and `[PL]`: standard modules, PCB standoffs/screws, stack screws, backplane fixture, header mating, servo connector, cap/parachute/servo, and battery fixture. | 100% inspection; no sampling. | Every modeled component and physical link is present, correctly located, and no unmodeled extra module type is introduced. |

Voltage drop is recorded. |
| SAA-T07 | Critical signals are measurable and fast lines remain electrically usable through the backplane. | Expected use case | Backplane with SPI/I2C/UART routes and all connected subsystem PCBs. | Model `backplane_limits`; NXP UM10204 I2C capacitance/rise-time guidance. | Inspect/testpoint accessibility; continuity/short test; measure I2C bus capacitance/rise time; scope SPI/UART/I2C under representative traffic. | 100% net continuity. For traffic, run ≥1,000,000 frames/messages with zero errors where automated logging is available. | All critical testpoints are accessible; no opens/shorts; I2C bus capacitance and rise time are within selected mode limits; zero protocol errors in the run. |
| SAA-T08 | Backplane PCB design is fabrication-ready. | Feared event: CAD/fabrication rule escape | Backplane CAD, Gerbers, BOM, and manufactured PCB. | Model `backplane_limits`; IPC design/acceptance literature. | Run ERC, DRC, and manufacturer DFM checks; inspect layer count, annular rings/clearances, footprints, silkscreen labels, pin-1 marks, and testpoint labels. | 100% rule-check review before fabrication and again on release candidate. | Zero ERC/DRC errors, no unreviewed high-severity warnings, two-layer PCB confirmed, footprints match 2 mm 2×6 headers and servo connector. |
| SAA-T09 | 3D-printed structural parts are produced with the intended process capability. | Expected use case | Printed standard modules, parachute/top module, and representative print coupons. | Model `assembly_limits`; ISO/ASTM 52902 AM geometric capability; FDM infill literature. | Inspect slicer/profile for 0.2 mm precision and 15% infill; print a benchmark/coupon set and measure critical features, screw bosses, PCB pockets, and backplane slot. | At least 3 coupons or 3 representative modules per printer/material batch; guard-band dimensional errors. | Slicer settings match model; critical feature error + U95 ≤ 0.2 mm unless drawing tolerances state otherwise; no warping that affects fit. |
| SAA-T10 | Self-tapping screw assembly remains serviceable during integration and maintenance. | Expected use case / feared event: stripped screw bosses | Standard stack modules and parachute/top module with flight screw types. | Model `assembly_limits`; FDM shell/infill screw anchoring guidance. | Verify no more than two self-tapping screw types; torque-limited assembly; repeated assemble/disassemble cycle test on representative holes. | Workshop screen: 10 cycles per screw type on representative holes. | SAA-T11 | Stack, backplane, PCB, and battery fixtures survive launch/transport vibration and 20G acceleration screening. | Feared event: loosening or fixture failure | Fully integrated v1.0 assembly. | Andøya 20G CanSat requirement; NASA SSRI/GEVS vibration workmanship guidance; CanSat environmental test template. | Pre-inspect, apply low-level sine sweep to find resonances, then random vibration or equivalent transport/fixture screening in three axes; if shaker unavailable, perform 20G-equivalent static/proof-load checks on heavy retained items. | 3 axes. | SAA-T12 | CanSat survives landing impact without structural or integration failure. | Feared event: landing shock | Fully integrated v1.0 assembly, powered off or safe mode as appropriate. | Model `envelope_limits`; ASTM D5276 free-fall drop method; CanSat environmental test template; ESA safe landing context. | Drop-test using a defined mission-equivalent height/surface/orientation set. | SAA-T13 | Parachute/top module opens without S&A interference. | Expected use case / feared event: deployment blocked | Parachute/top module with cap, parachute, servo, backplane clearance, and standard stack installed. | v1.0 parachute module physical components/links; ESA safe landing context. | Mechanical deployment dry-runs in relevant orientations before and after vibration/drop screens; record cap motion, clearance to backplane, parachute extraction path, and reset effort. | Qualification: 22 independent deployments with zero snags for R90/C90. ## Evidence required per test

Each test record should include: model baseline (`v0.1` or `v1.0`), as-tested configuration, operator, date, calibrated instrument IDs, raw measurements/logs, photographs/video where applicable, pass/fail rationale, deviations, anomalies, and links to the relevant model elements.
