# DPS v0.2 MBSE Views

DPS v0.2 delivers only the ground-station PCB hardware from DPS v1.0:

- `[PC] Ground Station PCB`
- `[PC] RFM95W LoRa`
- `[PC] XIAO ESP32`
- PCB footprints and SPI + 5V power traces between the two mounted PCs

There are no logical components, functions, or functional-chain views in v0.2.
