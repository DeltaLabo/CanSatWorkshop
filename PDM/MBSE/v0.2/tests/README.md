# PDM v0.2 IVV testing plan

This v0.2 testing plan was created because no `PDM/MBSE/v0.2/tests/README.md` existed at orchestration start.

- **Version gate:** PDM v0.2 -> PDM v1.0.
- **Selected detailed modeled activity for this session:** `PDM-V02-FC-BLE-DEPLOY` — BLE/manual parachute deployment verification definition.
- **Modeling rule:** PDM v0.2 intentionally covers manual BLE deployment only; no IMU/altitude fallback is present in this baseline.
- **Evidence convention:** execution evidence and reports should be archived under `PDM/MBSE/v0.2/tests/results/<verification-activity-id>/` unless a campaign-level results folder is later agreed.
